  • Q : 通知书什么时候能到手呢？

    A : 一般来说华科的通知书发得都算是比较慢一点的，而且越接近湖北的地方以往发得越晚。但是肯定能在开学之前到你手上啦，不必太过焦急误了假期。

    另外，通知书里面有入学须知，也会附送一份华科的周边纪念品哦。

  • Q : 有开学考试吗？

    A : 有三种考试 :

    1. 英语考试 : 仅关系到后面英语课程的分班教学

    考试内容 : 参照四六级

    考试难度 : 难度高于四级低于六级

    这个考试会分出前六十名分到英语托福班和雅思班的教学中，进了托雅班就不用学习大学英语（当然还是要考试）。

    所以虽然难度是四六级之间，但是要分到托雅班里还是需要比较高的英语应试水平的。

    2. 学生守则 : 很随意的考试，基本都会提前露出答案，或者考试时候也可以偷偷看书的。

    相信各位对于该做什么不该做什么有个基本的判断。也可以翻一翻学生手册看看自己在雷区蹦迪的舞技如何。

    3. 启明考试 : 这个暂时先不讲，详细看下面关于启明的解释。

    PS:除了学生守则考试需要达标以外，其他两门属于交白卷也没关系的考试。毕竟是选拔性的。

  • Q : 军训有多久呢？

    A : 在新生报道后几天开始（具体参照届时通知），按照惯例军训三周。

    注意军事训练的学分是必修学分，不及格来年重训。

  • Q : 启明学院是什么？

    A : 启明学院是实验班，有优点有缺点。

    优点 : 教学资源更强，学风更好。

    缺点 : 学习压力大，考核难度高，成绩偏低，并且新的实验性课程不一定好。

  • Q : 进入启明的方法？

    A : 有两种方法 :

    1. 填选志愿时，有的可以直接选择启明班。

    2. 在英语入学考试时那几天会有启明的选拔考试，考试在军训期间

    考试内容涉及物理和数学 高中竞赛难度。

    还有一些综合题目，可能涉及经济、政治等等方方面面。

    还有不是每个专业都有对应的启明班

    启明官网 : http://qiming.hust.edu.cn/

  • Q : 华科可以转专业吗？转专业难吗？

    A : 12月左右考试 期中再考 半年左右

    有两次转专业机会 大一随便转 大二院内转（其实大二也能转院。。不过比较麻烦。。）

    当然也可以降级转专业，可以转到绝大多数专业。

    考试包含笔试加面试 笔试考微积分和英语，文科还会考作文什么的，跟具体专业有关

    平时好好学习+认真预习目标专业知识+面试稳定问答几乎可以保证转专业成功。

    有的专业较难转入 比如电信 电气 光电等 只能转入中英班（中法班） 转入需要缴纳一定费用

  • Q : 大一可以带电脑么？

    A : 基本上是不让带的，但是大部分专业你带了也没人管，少数专业管得严。

    学校是有机房的，总体来说大多数人带电脑的用途主要不是学习。

    所以如果你自控能力很差，那不妨不要带。

  • Q : 大类专业是什么意思啊？

    A : 绝大多数大类专业都是大一下学期分流，具体就是大类方向下的一些细支专业。

    比如计算机大类下的软件工程和网络空间安全明显不是一个方向的。

  • Q : 假期需要提前学习什么课嘛？

    A : 不需要，但是可以提前学习。从成绩的角度来说，假期学习微积分是最划算的。

学校生活

  • 吃 :

    校内有三四十家食堂，各种口味都有。

    总体可能比较偏咸和油一些，毕竟是大锅菜。

    学校西边出去就是光谷，有步行街和世纪城两个大商圈，好吃的地方还是不少。

  • 住 :

    四人间 上床下桌 配备空调

    少数楼层配备公共洗衣机 大多数宿舍独卫

    热水夏天早中晚供应冬天中午没有

    学校分冬令时和夏令时，以五一到十一为夏令时，其余时间冬令时

    夏令时下午以后的时间安排会延后半小时并且不停电，冬令时工作日晚23:00停电。

    工作日晚0:00断网，周末不断网，期末不断电。

  • 行 :

    校内有校车，一块钱。

    有三条路线，韵苑->紫淞，韵苑->南大门，韵苑小区->南大门。

    校车小且少而华科大，大多数人还是使用电动车和自行车通行。

    学校外部南大门、南三门、南四门都有毗邻的地铁站很方便出行。

  • 衣 :

    华科没校服，随便穿。

    但是武汉冬天很冷，夏天很热，请带好冬夏衣物。

  • 天气 :

    晴天骄阳，雨天暴雨。也可能上午晴天，中午吹风，下午暴雨。

    反而会让天气好的时候心情特别好吧。

  • 学生会 :

    学校里有团学联和资助委员会，团学联比较接近传统的学生会。

    资委一部分职能和团学联的部门重复，另外资委和资助的钱打交道比较多。

    有行政部，宣传部，新闻部，文艺部，阳光俱乐部等等。。

    另外，资委和团学联分别只能加一个部门。

    就是说你可以团学联和资委都报，但是最后资委只能去一个部门，团学联只能去一个。

    不过团学联里的某些部门会因为你班级里的职位而强制加入。

    比如心理委员会加入心理部等等。

  • 社团 :

    华科内有很多社团，但是大部分比较水。

    军训中期会有“百团大战”，有兴趣的同学到时候记得不要错过。

  • 技术组织 :

    华科内技术组织比较多，有比较出名的dian，联创和冰岩等。

    这些组织基本上都聚焦在计算机技术或者互联网产品上，有兴趣的同学可以具体了解。

一家之言，听听即可

每年，都有着众多的高三学子，褪去高中生的外壳，满心喜悦，满心憧憬的，望向前方。

高中的三年，有的人过得浑浑噩噩，有的人过得狂放不羁。

昨日的你们，还沉浸在题海中，还紧张的对着答案，为了考试而欢笑，而悲伤。

但是，过去已经过去，你们即将迎来未来。

高中的老师会对你们说，你们现在辛苦了，上了大学就可以好好玩啦。

这句话，对吗？对，也不对。

大学是一个自由的地方，你可以做很多事情。

你可以去尽情的谈一场充满着浓郁青春气息的恋爱。

你可以加入各种社团，挥洒自己的青春和汗水。

你也可以沉浸在图书馆，与书本做伴，探索这个无穷的世界。

你也可以加入一个技术组织，每天和朋友一起写代码，做硬件。

每个人都可以选择自己喜欢的生活方式。

高中教会了你们很多，但是，往往忘记了一件事。

忘记了让你们知道自己想做什么。

很多人来到了大学会感到强烈的迷茫，以及对未来的恐惧。

大学给了你自由，但是，你却不知道怎么向前走。

很多人每天就是上课，回宿舍，吃，玩，睡觉。

然后，当他们要离开这个地方的时候，发现自己根本没准备好。

四年，多么宝贵的时间。

如果你有想做的事情，请尽情去做。

如果你还在迷茫，请尽情去尝试。

如果不管怎么做都会后悔的话，那就选择后悔的少的一方吧。

珍惜现在，趁你还能选择。

有的人害怕自己的选择错误。

比如说，自己想做的事情做的人不多，而且很难找工作。

这让我想起来一件事情。

大家小时候想，去清华呢，还是去北大呢？

后来发现自己都没法去。

但是我们不是只有清华和北大。

工作不好找，总会有工作，没有大公司总有中小公司。

与其去不停的迷茫，不停的担心，不停的想。

少想多做，知行合一。

人生的选择很多，请选择一条让你最不会后悔的路。

假如你选错了，也不要担心，我相信，只要一直向前走，就会有好事发生。

这个世界上，最美的事情往往不是心想事成，而是阴错阳差。
